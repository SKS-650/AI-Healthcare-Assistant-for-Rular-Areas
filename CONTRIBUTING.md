# Contributing

Thank you for contributing to AI Healthcare Assistant for Rural Areas.

## Getting Started

1. Fork or clone the repository.
2. Create a feature branch.
3. Keep changes focused and documented.
4. Run relevant tests before submitting changes.

## Project Areas

- `mobile_app/` contains the Flutter frontend.
- `backend/` contains the FastAPI services.
- `ai_models/` contains AI/ML model code and experiments.
- `admin_dashboard/` contains the web dashboard.
- `documentation/` contains project documentation.
