# AI Models

AI and machine learning workspace for the AI Healthcare Assistant for Rural Areas project.

## Areas

- `datasets/` stores raw, processed, training, validation, and testing datasets.
- `preprocessing/` and `feature_engineering/` prepare features for training.
- `training/`, `evaluation/`, and `inference/` contain the model lifecycle.
- `prediction/`, `emergency_detection/`, `chatbot/`, `nlp/`, `llm/`, and `speech/` contain domain-specific intelligence modules.
- `saved_models/` stores model artifacts.

## Run

```bash
python main.py
```
